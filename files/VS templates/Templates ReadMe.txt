To use templates, copy them to the CSharp project templates folder of your Visual Studio 2008 installation, for me this folder is:

C:\Program Files (x86)\Microsoft Visual Studio 10.0\Common7\IDE\ProjectTemplates\CSharp\Windows\1033

Then go to start menu and open a Visual Studio command prompt and at the prompt run:

devenv /installvstemplates

Next time you run Visual Studio and create a new C# project you will have new choices:

UO Machine GUI Script and UO Machine Standard Script, remember to add a reference to UOMachine.exe after you create a new project.
